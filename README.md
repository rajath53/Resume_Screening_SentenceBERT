# Resume Screening and Candidate Ranking System

## Project Overview
This project automates resume screening
using NLP and Sentence-BERT.

## Features
- Resume Upload
- Resume Parsing
- NLP Preprocessing
- Skill Extraction
- Candidate Ranking
- Result Visualization

## Technologies
Python, NLP, Sentence-BERT,
Scikit-learn, Pandas,
Matplotlib

## Workflow
Resume -> Parsing ->
Preprocessing ->
Sentence-BERT ->
Cosine Similarity ->
Ranking

## Output
Ranked candidate list
with match score
and skill matching.
